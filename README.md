# Reproducibility Guide

Welcome to the **Reproducibility Guide**! This project is designed to help researchers and students (especially in Psychology) implement open science practices and reproducible workflows.

## View the Website
The live version of this guide can be found here: 
[https://schillinglars.github.io/reproducibility_guide/](https://schillinglars.github.io/reproducibility_guide/)

## Project Structure
- `content/`: The Quarto source files for the different chapters.
- `docs/`: The rendered HTML files for the different chapters (hosted via GitHub pages).
- `data/`: Sample datasets used on the website.
- `images/`: Images used on the website

## How to Contribute
This is an open educational resource. Feel free to fork the repository or submit a pull request if you find typos or want to suggest improvements.
